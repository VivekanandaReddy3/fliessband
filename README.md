

# SalesLytic

## 📝 Description
SalesLytic is a business analytics tool that converts complex sales data into actionable visual dashboards using SQL and Tableau. It offers insights into sales performance, customer behavior, and top-performing products.

## 🚀 Features
- Sales by region and product
- Revenue and customer insights
- Interactive dashboards for business use

## 🛠️ Tech Stack
- Data: SQL (PostgreSQL/MySQL)
- Visualization: Tableau

Gained hands-on experience with data wrangling and visualization tools to support business decision-making.
